# 💊 Pills Reminder App

A Java Swing desktop application that helps users manage and track their daily medications.

## Features
- Add medications with name, days, and pill count
- View medication history
- Increase day counter for each medication
- Change background color theme (Light Blue, Pink, Purple)

## How to Run
1. Open the project in VS Code
2. Make sure you have JDK installed
3. Run `MainMenuWindow.java`

## Project Structure
- `MainMenuWindow.java` – Main menu UI
- `AddMedicationWindow.java` – Add new medication
- `MedicationHistoryWindow.java` – View medication history
- `medications.txt` – Stores medication data
